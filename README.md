# Task Tracker (SQR Project)

Small educational task tracking app built for SQR quality-gates practice:
- Backend: FastAPI
- Database: SQLite
- Frontend: Streamlit
- CI: GitHub Actions quality gates

## Features
- Create, list, search, update, complete, and delete tasks
- SQLite persistence
- OpenAPI docs auto-generated by FastAPI
- Streamlit demo UI
- Automated tests and quality checks

## Limitations (intentionally out of scope)
- No authentication/authorization
- SQLite only (no production database)
- No distributed deployment
- Minimal UI and no advanced frontend design
- No hard real-time performance guarantees

## Demo smoke-test path
1. Create a task from Streamlit.
2. Confirm it appears in list.
3. Toggle completion status.
4. Filter to Completed/Open and verify behavior.
5. Search by keyword.
6. Delete task and confirm it disappears.
